SPMS UI (Static Bootstrap Pages)

Location:
  midterm-project/public/spms/

Pages:
  /spms/index.html         - Dashboard
  /spms/students.html      - Student List (search + filters)
  /spms/add-student.html   - Add Student form (image preview)
  /spms/profile.html       - Student Profile (cards per section)
  /spms/reports.html       - Reports generator (filters + CSV export + print)

How to view:
  Option A (Vite dev server):
    - Open a terminal in: midterm-project
    - Run: npm install
    - Run: npm run dev
    - Visit: http://localhost:5173/spms/index.html

  Option B (Open directly):
    - You can open the HTML files directly in a browser.
    - Note: Some browsers may restrict local file access; using Vite is recommended.
